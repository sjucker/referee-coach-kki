 Manual

Start Video Report

Copy the game number from Basketplan. If the game has the video-icon set, it will automatically be used, otherwise you will have to provide the YouTube-URL yourself.

Paste the game number into the input-field and press "Search".

Select the referee you want to review and press "Start"

If no YouTube-video was found in Basketplan, you will have to paste the URL into the field. Just copy the URL of the specific YouTube-video.


Creating the Report

At any time you can save the current state of the report. If you return to the homepage, you can edit it again by pressing the edit-icon.


Creating Video Comments

The corresponding YouTube video is displayed. Start the video. Once you found the beginning of a scene you want to comment on, pause the video and press the button "Create comment for current timestamp"). At the right side you can now add the comment for this scene.

Finishing the Report

Once you confirm to finish the report, the report will become read-only and the reviewed referee will receive an email containing the link to this report (same as pressing on the icon next to the finish-icon).

Copying a Report

Often, you want to re-use the same scenes commented on in another video report about the same game (but other referee). In this case, use the copy functionality: all video comments will be copied into the new video report for the other referee. Then you only need to adjust the comments for the current referee, but you do not need to search the same scenes in the video again.

Deleting a Report

As a non-admin user, you are allowed to delete your own video reports, as long as it is not yet finished (i.e., the email was not yet sent to the referee).
